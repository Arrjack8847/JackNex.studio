# JackNex Studio — Personal Portfolio Website

A personal creative developer portfolio built to showcase web development projects, digital experiences, design work, and selected case studies.

## Live Demo

[Visit JackNex Studio](https://jack-nex-studio.vercel.app/)

## Overview

JackNex Studio is a single-page personal portfolio website designed to present my work as a frontend developer and creative builder.

The website combines a clean editorial layout with animated sections, project highlights, process information, personal introduction, social links, and a contact call-to-action. It is designed to give visitors a clear overview of my skills, work style, and selected projects.

## Features

* Responsive portfolio website for desktop and mobile
* Hero section introducing the portfolio
* Featured work showcase
* “How I Work” process section
* Statistics and highlights section
* About section
* Contact call-to-action section
* Persistent social sidebar
* Animated global background effects
* Smooth motion and interaction effects
* Client-side routing with a fallback 404 page

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion
* React Router
* Lucide React
* Vitest
* Vercel

## Project Structure

```text
src/
  components/       # Reusable page sections and interface components
  pages/            # Application pages and fallback route
  assets/           # Static project assets
  App.tsx           # Application shell and routing
  main.tsx          # Application entry point

public/             # Public static files
```

## Local Setup

Clone the repository:

```bash
git clone https://github.com/Arrjack8847/JackNex.studio.git
cd JackNex.studio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local development site:

```text
http://localhost:5173
```

## Available Scripts

```bash
npm run dev         # Start the Vite development server
npm run build       # Create a production build
npm run build:dev   # Create a development-mode build
npm run lint        # Run ESLint checks
npm run typecheck   # Run TypeScript type checking
npm run test        # Run the Vitest test suite
npm run test:watch  # Run tests in watch mode
npm run preview     # Preview the production build locally
```
